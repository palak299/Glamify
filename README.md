# ✨ Glamify – Modern E-Commerce Website

🛍️ **Glamify** is a modern, stylish, and fully responsive e-commerce website designed to provide users with a seamless online shopping experience.  
This project focuses on creating an attractive user interface with smooth navigation, interactive sections, and a clean shopping layout.

Glamify is built using front-end web technologies and showcases how an online shopping platform can look visually appealing while remaining user-friendly and responsive across all devices.
